anchor tag

< a >- Anchor : HTML anchor tag define a hyperlink that links one page to another page.

below are attributes used in anchor tag :

  • Href - its used to provide links to connect
  • Target- where should open the link
    • _blank- its opens new tab of brower
    • _self- opens the linked Document in the same frame as it was clicked
    • _parent- opens the linked document in the same freame
    • _top- opens the linked document in the full body of the window
    • frame name- opens the linked document in the named iframe

external link

whwn we connect external sites on our web pages.

E.x : facebook , twitter , whatsapp , etc then it is called external link

facebook | twitter | whatsapp

Internal or Multi Page Appication

when we link our one page to another page then it is called Multi page Appication or Internal linking.

example with code :

list | list2 | single page
