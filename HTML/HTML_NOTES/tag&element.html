HTML TAG :

  • html tags are like keywords which defines that how web browser will format and display the content
  • all html tags must enclosed with < > these brackets
  • every tag in html perform different tasks
  • if you have used an open tag < > then you must use a close tag < />
  • unclosed html tags
    • < br > tag : br stand for break line , it breaks the line of the code.
    < hr > tag : hr stand for horizontal rule. this tag is used to put a line across the webpage

    HTML ELEMENT :

  • an element in html usually consists of a start tag < tag name > close tag < /tag name> and content inserted between them.

    • e.g.

    < p > welcome < /p >

    there are two types of element

  1. block-level element
  2. inline element

  1. block-level element : a block- level element always start with new line and takes the full width of web page, from left to right

    2. Ex :

    < address > , < article > , < aside > , < blockquote > , < canvas > , < dd > , < div> , < dl > , < dt> , < footer > , < form > , < h1 > to < h6 > , < header > , < hr > , < li > , < main > , < ol > , < p > , < section > , < table> , < ul> , < video> , etc......

  2. inline element : these elementdoes not start with new line and take width as per requirement.
    3. the inline element are mostly used with other element.
    Ex :

    < a > , < b >, < abbr >, < bdo >, < br >, < button > , < cite >, < code >, < dfn >, < em >, < i >, < img >, < map > < object >, < input > ,< samp > ,< select > ,< span >,< strong >,< sub >, < time >, < tt >, < var >, etc......
